# 🩺 Pneumonia Detection from X-Ray

This project uses Deep Learning to detect pneumonia from chest X-ray images.

## 🚀 Features

* Image classification using CNN
* Trained on 7000+ X-ray images
* Predicts Pneumonia vs Normal

## 📦 Installation

Clone the repo:

```bash
git clone https://github.com/absarzaidi/Pneumonia.git
cd Pneumonia
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## ▶️ Run the project

```bash
python pneumonia_detection.py
```

## 📁 Project Structure

* pneumonia_detection.py → main model script
* data/ → dataset (not included)

## ⚠️ Notes

* Dataset is not uploaded (too large)
* You need to add your own dataset
